================================================================================
  ⚡ THE 5-MINUTE QUICK START
================================================================================

1. Create GitHub repository:
   - Go to https://github.com/new
   - Name: "neurovision"
   - Uncheck "Initialize with README"
   - Click "Create repository"

2. Copy your repository URL:
   https://github.com/YOUR_USERNAME/neurovision

3. Run these commands in this folder:

   git init
   git remote add origin https://github.com/YOUR_USERNAME/neurovision.git
   git add .
   git commit -m "Initial commit: Add Neurovision brain visualization tool"
   git branch -M main
   git push -u origin main

4. When prompted for password, use a Personal Access Token (see guide)

5. Verify at your GitHub URL ✅
